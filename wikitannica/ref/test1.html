1. Connect to CyberGhost VPN

sudo cyberghostvpn --connect --country-code LT

2. Install .deb Package

sudo dpkg -i /item

3. Get Rid of Screenshot Sound

sudo mv /usr/share/sounds/freedesktop/stereo/camera-shutter.oga /usr/share/sounds/freedesktop/stereo/camera-shutter-disabled.oga

More details can be found on Ask Ubuntu.

4. Restore a Git Repository

git reset --hard origin/branch

5. Merge File with rsync

rsync -av z1/ z2/ ---puts z1 contents into z2

6. Get Rid of Terminal Username

nano ~/.bashrc

Add the following line:

export PS1="\W \$"

Then press Ctrl + X followed by Y to save and exit.

7. Find All Directories with Less Than 50 Files

find . -type d | while read dir; do [ $(find "$dir" -mindepth 1 -maxdepth 1 -type f | wc -l) -lt 50 ] && echo "$dir"; done
